A homepage about @Taranakineko, use [zdhxiong / mdui](https://github.com/zdhxiong/mdui/) 🚧

大概是半手搓的，代码写得有点渣且乱 ~~，连 commit 信息都不写好，~~ 见谅（）

---

## Start 开始

安装 rollup 用于压缩

```bash
npm install --global rollup
```

安装依赖

```bash
npm intsall
```

生成 main.js

```bash
npm run build
```

or

```bash
rollup -c
```
